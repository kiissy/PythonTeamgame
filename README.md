# RunAway

-----------
산더미 같이 놓인 과제를 하기 위해 오늘도 컴퓨터 앞에 앉은 당신이지만, 오늘따라 키우고 있던 강아지와 고양이의 눈빛이 심상치 않습니다.

산책을 나가자고 조르는 강아지와 키보드 위에 턱 하니 자리를 차지하려고 고군분투하는 반려동물들을 피해 과제를 완수해낼 수 있을까요?

끊임없이 당신을 향해 돌진하는 반려동물들의 반란을 극복하세요!
